# Physics Course Assets
This repository hosts texture and cubemap assets used for educational physics demos (WebGL) at Ben-Gurion University.

**Attribution:**
Assets sourced from [Shadertoy.com](https://www.shadertoy.com) hosted here to facilitate cross-origin resource sharing (CORS) for educational demonstrations within the university's learning management system.

**Usage:**
These files are intended for non-commercial, educational use only.
